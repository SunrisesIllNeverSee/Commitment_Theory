# inbox

Drop zone for all new files. Everything enters here first.

**Processing flow:**
1. Drop file in `inbox/`
2. Identify which track it belongs to (CT, Legal_Theory, MISC)
3. Move to the appropriate destination:
   - Research/reference docs → `[Track]/workspace/source-docs/`
   - Paper drafts → `[Track]/papers/[paper-folder]/`
   - Context/analysis → `[Track]/workspace/context/`
4. Nothing stays in inbox permanently
