# academic-research

Central hub for all academic writing pipeline work on the Commitment Theory publication program.

## What lives here

| Folder / File | Contents |
|---------------|----------|
| `CLAUDE.md` | Full CT framework context + pipeline entry points for every paper. Read this first in any writing session. |
| `ACTION_PLAN.md` | Consolidated review findings + prioritized action list for all 34 papers. |
| `pipeline/[paper-id]/` | Academic pipeline outputs per paper — material passports, research reports, drafts, reviewer reports, revision history. Created when work begins on a paper. |

## How to start writing a paper

1. Open this folder in a Claude Code session
2. Read `CLAUDE.md` — it has everything you need to orient
3. Say: `"I want to write [Paper ID] — [Title]"` → `academic-pipeline` launches
4. Or say: `"I already have a plan for [Paper ID], review it"` → enters mid-pipeline at integrity check

## Pipeline outputs go in
`pipeline/[paper-id]/` — e.g., `pipeline/CL-001/`, `pipeline/P-000/`, `pipeline/L-002/`

## Priority order (from ACTION_PLAN.md)
1. P-000 — deposit (no writing needed, just format + submit)
2. CL-001 — write now (data complete, no blockers)
3. IS-001 — write now (existing material maps directly)
4. L-000 — CCT→CT update + deposit
5. L-002 — corpus planning (blocks 4 other legal papers)
